## 🚀 Key Features
- **Code Submission:** Users paste code snippets and receive instant quality ratings.
- **Roast Mode:** An optional toggle to activate maximum sarcasm for a more entertaining code analysis.
- **Detailed Analysis:** Specific feedback highlighting what is wrong (and right) with the code, categorized by severity levels (critical, warning, good).
- **Fix Suggestions:** Visual diffs demonstrating how the code could be improved applying best practices.
- **Shame Leaderboard:** A ranking of the "worst code on the internet," ranked by shame/score, allowing users to see how they compare.
