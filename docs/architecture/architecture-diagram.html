DevRoast Monorepo Architecture

React + Spring Boot platform for code-roast analysis with OAuth2/JWT identity, PostgreSQL persistence, and Ollama-backed LLM inference via Docker Compose.

Application Edge

  • • Browser users hit frontend on 5173 (dev) or 4173 (prod-like).
  • • React 19 UI handles register/login/logout and session retrieval via /me.
  • • Frontend routes auth and API traffic to backend using Vite proxy/env config.
  • • GitHub OAuth entry starts from frontend and returns through backend flow.

Core Services

  • • Spring Boot 4 API exposes auth/session + code-analysis endpoints.
  • • Security stack includes OAuth2 Authorization Server, OAuth2 Client, JWT setup.
  • • AI service (BashCode + HttpClient) invokes Ollama /api/generate with codellama.
  • • Backend persists submissions, analyses, fix suggestions, and user records.

Data & Operations

  • • PostgreSQL 16 stores operational data and identity data.
  • • Flyway migrations (V1-V3) apply deterministic schema evolution.
  • • Docker Compose orchestrates frontend, backend, postgres, and ollama services.
  • • External integrations are GitHub OAuth2 and self-hosted Ollama only.
